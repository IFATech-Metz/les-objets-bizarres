Notice utilisation du catalogue


  • - installer wamp sous windows ou mamp , xamp lien sous mac o.s 

  • -lancer xamp qui sera normalement sur le bureau :

  • -lorsque la fenêtre de configuration s ouvre activer apache et mysql en appuyant sur start.

  • -copier le dossier les objets bizarres C:\xampp/htdocs .

  • -aller sur la page localhost/phpmyadmin.

  • -si un login et mot de passe sont exigés login:root,password:vide.

  • -une fois connecte a phpmyadmin cliquer sur nouvelle base de données située en haut a gauche de l’ écran.

  • -dans le premier champ ou l’ on vous demande le nom de la base de données écrire : catalogue.

  • -ensuite appuyer sur le bouton créer.

  • -une fois la base de donnees creee appuyer sur l onglet importer situes tout en haut au centre de l’ écran.

  • -ensuite appuyer sur le bouton parcourir allez a l emplacement du site c est a dire : c:\xampp\htdocs\les-objets-bizarres.

  • -et sélectionner le fichier catalogue.sql.

  • -une fois ceci fait appuyer sur exécuter situe tout en bas de la page.

  • -désormais le site est fonctionnel.

  • -ouvrez le navigateur et entrer l’ adresse suivante : localhost/lesobjets-bizarres.
